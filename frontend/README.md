# Frontend

Not yet implemented. Scaffolded folder structure only:

- `src/components/` - reusable UI components
- `src/pages/` - route-level views (dashboards, login, settings)
- `src/services/` - Axios API clients
- `src/hooks/` - custom React hooks
- `src/contexts/` - React context providers (auth, theme)
- `src/layouts/` - page layout shells
- `src/assets/` - static assets
- `src/styles/` - global styles/themes
- `src/utils/` - shared frontend utilities

This will be built out in the Frontend phase, once the backend REST API surface
(beyond authentication) exists to consume.
